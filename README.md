# BlueBot | For Discord
Hi, we have created a special bot on Discord that will help you moderate and improve it.
For example, there are tips on how to make a better server.
## Instructions for adding a bot.
1. Go to the Discord developer portal.
   - Log in to the site with your Discord account.
   - Create a new bot.
   - Copy the bot token.
2. Go to the config.json file and paste it into the section called token
3. Configure the config.json file, enter, for example, the name of the server, follow the instructions in the file.
4. Open the bot using python
### Note: install discord.py
